Reports and Export

Where to export plots Use the Predict tab, then open the highlighted Plots & Reports box under the sequence form. Use Download Prediction Plot Report ZIP for screening plots, or Download Variant Plot Report ZIP after choosing a variant utility.
Utility report packs Use the Utilities tab and the highlighted Plots & Reports box to export descriptor plots without running the antifungal model.
Accepted formats Use one sequence per line, comma/semicolon separated text, or FASTA records. Invalid sequences are reported per record so the full batch can still be reviewed.
CSV, FASTA, plots, and reports Prediction exports include scores, rank, duplicate status, screening tier, formula, pI, charge at pH 5.5/7.4, descriptors, interpretation flags, validation errors, clean FASTA output for valid sequences, editable SVG plots, and a self-contained HTML report. Variant report packs add parent ID, mutation, position, score delta, rationale, and descriptor fields.
Model representation D-amino acids, cyclization, terminal capping, non-canonical residues, salts, and formulation effects are not represented in the sequence model.
Large-screening workflow Paste or upload candidate libraries as FASTA/text, rank by AFPRED probability, retain candidates marked advance/review, then run mutation utilities only on selected parents before toxicity, hemolysis, novelty, synthesis, and experimental triage.

In-app Workflow

1. Predict Load a peptide library in the Predict tab and rank candidates with AFPRED.
2. Calculate Use the Utilities tab for PepCalc-style physicochemical calculations without invoking the model.
3. Design Run mutation analysis only for selected high-priority parent peptides.
Developer endpoints JSON endpoints remain available at /api/predict, /api/variants, and /api/properties for automation, but normal use does not require them.
